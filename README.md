# Pig, Pug, or Plopp? 🐷 🐶  
This is a custom neural network built without deep learning libraries like TensorFlow or PyTorch.

## ⭐ It works like this  
Architecture: 784 Input nodes (28x28 pixels) → 100 Hidden nodes (ReLU) → 3 Output nodes (Sigmoid).

Optimization: Stochastic Gradient Descent with Backpropagation.


## 🎯 Educational Goal  
Usage: For a junior to learn ML in a cute-ish way. 

Aim: To make a senior go "Aww.."


## 📈 Current Milestones  
Iteration 1: Successfully trained on a pool of 15 images.

Convergence: Reduced training loss from 11.26 to ~0.11 over 500 epochs.

Weight Persistence: Includes a system to save/load trained weights (.npy files) to avoid retraining.
